# MedBrief – AI-powered Clinical Research App

## Overview

MedBrief is a mobile application designed to help doctors stay updated with clinical research through AI-generated summaries and audio briefs.

## Problem

Doctors often face:

* Limited time to read lengthy research papers
* Information overload
* Missing important clinical updates

## Solution

* AI-powered study summaries
* Personalized research feed based on specialty
* Audio-based learning
* Real-time alerts

## Work Done

* Designed complete **Information Architecture (Sitemap)**
* Created **end-to-end user workflows** (Auth, Onboarding, Research, Notifications)
* Built **wireframes for key screens**
* Defined **user persona** and **edge cases** to cover real-world scenarios

## Key Features

* Authentication (Google, Email, 2FA)
* Personalized onboarding
* Research search with filters
* Study insights & evidence score
* Audio playback
* Bookmark & notifications

## Outcome

* Structured a healthcare product from scratch
* Ensured clarity in user journeys and navigation
* Improved usability by considering edge cases and user behavior
