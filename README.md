# MedBrief – AI-powered Clinical Research App

## Overview

MedBrief is a mobile application designed to help doctors stay updated with clinical research using AI-generated summaries and audio briefs.

## Problem

Doctors face:

* Lack of time to read full research papers
* Information overload
* Missing important updates

## Solution

* AI-powered study summaries
* Personalized research feed
* Audio-based learning
* Real-time alerts

## Features

* Authentication (Google, Email, 2FA)
* Personalized onboarding
* Research search with filters
* Study insights & evidence score
* Audio playback
* Bookmark & notifications
